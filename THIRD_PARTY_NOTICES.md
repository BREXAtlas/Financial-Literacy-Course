# Third-Party Notices

This project currently uses no third-party image assets, icon packs, fonts, or chart libraries. All
visualizations are generated at runtime from plain HTML/CSS/SVG by `assets/visualization-engine.js` using
first-party curriculum data (`data/visualizations.js`) — there are no image files to license or attribute.

If original illustrations, icons, or a chart library are added in the future, list each one here with its
license, source URL, and the file(s) it applies to, per `docs/SOURCE_POLICY.md`.
